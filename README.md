# usingCMAP
Code and examples to use CMAP for data visualization\
Krista Longnecker 5 May 2026

See the script ```plotBIOSSCOPE_examples.ipynb```. This script is written in Python, but CMAP also supports R and MATLAB access to the database.

## Please note (this is important):
Note that to run the script you will need a folder labeled 'config' in your working directory. In that folder place a file called ```my_config.py```. I have placed an example version of that file here (```my_config_EXAMPLE.py```). Do NOT leave this in a place where it would be synced to GitHub or any other online repository.
